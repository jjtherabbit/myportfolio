Cat Story

The Cat Story project is an interactive web gallery for exhibiting my cats' photos. The main structure of the page uses the Bootstrap Grid System to create. The 3x3 grids present on the page are the pictures' frames. There is also a zoom-out picture frame that is created by adjusting its absolute position in CSS. (The picture frame always display in the center of the screen with a "p" tag as the close button) I used a hidden class(display: none;) to hide the zoom-out picture frame. It only shows up when the user clicks on the picture in the grids. This project helped me to learn how to use Bootstrap and Javascript DOM together. It also helped me to practice my CSS skills, such as filter and backdrop-filter.

Dice Games

This project is a mini-game for two players. The purpose of the project is to practice my problem-solving skill with Javascript. In the project, I programmed the functions for these three buttons: ROLL DICE, HOLD, and NEW GAME to make the game work. I created a random number on the back that ranges from 1 to 6 as the dice number. The ROLL DICE button receives the user instructions to roll the dice for each round. It also helps with switching players when the dice hits 1(By controlling the display of the blur layer on both sides to control the game rounds). The HOLD button saves the players' current number into the total score and switches players. The NEW GAME button resets all the elements on the page to start a new game. By doing this, it will boost the performance of the game.
