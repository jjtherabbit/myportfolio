Bootstrap Mock Website

The Bootstrap mock website is a project that uses Bootstrap, HTML, and CSS. In this project, the concept of responsive design has involved which the size of the web page and the components will always change to the best way to adapt the device screen size. The goal of this project is to imitate the Themes website from Bootstrap to its closest similarity. The techniques of Bootstrap used to build this website are the Grid system, flexbox, Collapse navbar, dual display of navbar, card, dropdown menu, and nav(footer). It supports most mobile devices in the market include Samsung S5, iPhone 6/7/8/x/plus, and iPad/iPad pro.
